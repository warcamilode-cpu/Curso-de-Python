estados = ["Aprobado", "Reprobado", "Aprobado",
           "Aprobado", "Reprobado", "Aprobado"]

conteo = {}
for estado in estados:
    conteo[estado] = conteo.get(estado, 0) + 1

print(conteo)

La línea clave — conteo.get(estado, 0) + 1:

Si el estado no existe aún → .get devuelve 0 → guarda 1
Si el estado ya existe → .get devuelve el conteo actual → le suma 1
